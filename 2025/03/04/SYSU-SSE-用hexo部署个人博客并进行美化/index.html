一、在 github.io 上用 hexo 部署个人博客

1. 准备工作

a. 安装Node.js和npm

Hexo是基于Node.js的，因此需要先安装Node.js和npm（Node.js的包管理器）。

下载并安装Node.js：https://nodejs.org/。

安装完成后，npm也会自动安装。

b. 安装Git

GitHub Pages需要通过Git来管理代码。如果尚未安装Git，请从以下链接下载并安装：https://git-scm.com/。

c. 注册GitHub账号

https://github.com/

2. 创建GitHub仓库

a. 登录GitHub

b. 创建仓库

点击右上角的“+”号，选择“New repository”。

仓库名称必须是<你的用户名>.github.io，例如：hhhhhhhhhxy.github.io。这是GitHub Pages的默认仓库名称。

选择“Public”（公开）。

不需要初始化仓库，直接点击“Create repository”。

3. 安装Hexo并初始化博客

a. 安装Hexo

打开终端（Windows用户可以使用Git Bash或CMD），运行以下命令安装Hexo：

npm install -g hexo-cli

b. 初始化博客

选择一个本地目录作为博客的根目录，运行以下命令：

hexo init blog

进入该目录：

cd blog

c. 安装依赖

在博客目录下运行以下命令安装依赖：

npm install

4. 配置Hexo

a. 编辑配置文件

打开_config.yml文件，这是Hexo的配置文件。主要需要修改以下内容：

#网站标题title: 你的博客标题

#作者author: 你的名字

#网站描述description: 你的博客描述

#网站URL（GitHub Pages的地址）url: https://<用户名>.github.io

#部署配置deploy:

type: git

repo: https://github.com/<你的GitHub用户名>/<你的GitHub用户名>.github.io.git

branch: main

注意：<你的GitHub用户名>替换为你的GitHub用户名。

5. 生成博客并部署

a. 生成静态文件

在博客目录下运行以下命令生成静态文件：

hexo generate

b. 部署到GitHub Pages

 安装hexo-deployer-git插件

在你的博客目录下运行以下命令安装该插件：

npm install hexo-deployer-git –save

–save选项会将插件添加到package.json文件中，方便后续管理。

运行以下命令将博客部署到GitHub Pages：

hexo deploy

如果部署成功，你会看到类似以下的输出：

INFO Deploy done: git

6. 访问你的博客

部署完成后，访问https://<你的用户名>.github.io即可看到你的博客。

二、美化博客页面

1. 选择合适的主题

Hexo提供了丰富的主题，这些主题可以快速改变博客的外观。你可以根据自己的喜好选择一个主题，或者对现有主题进行自定义。
查找主题：访问Hexo官方主题列表，浏览并选择一个喜欢的主题。

安装主题：

在博客目录下运行以下命令安装主题：

npm install hexo-theme-<主题名> –save

例如，安装hexo-theme-butterfly：

npm install hexo-theme-butterfly –save

在博客主文件夹创建_config.butterfly.yml文件，进入node_modules,找到hexo-theme-butterfly文件夹，将_config.yml文件复制到博客主文件夹的_config.butterfly.yml；修改主题配置文件_config.yml，将theme字段设置为新主题的名称：

theme: butterfly

运行下面命令后可在本地访问：

hexo cl

hexo g

hexo s

2. 自定义主题

如果你对现有主题不满意，可以通过自定义CSS和HTML来调整页面样式。

修改主题配置文件：许多主题提供了自己的配置文件（如themes/landscape/_config.yml），你可以在这里调整主题的颜色、字体、布局等。

添加自定义CSS：

在主题目录下找到source文件夹，如果没有则创建一个。

创建一个css文件夹，并在里面添加一个custom.css文件。

在主题的layout文件夹中找到layout.ejs或layout.swig文件，添加以下代码以引入自定义CSS：

<link rel="stylesheet" href="/css/custom.css">

3. 添加自定义页面

你可以创建一些自定义页面，如“关于我”、“项目展示”等。

创建页面：

hexo new page “about”

这会在source目录下创建一个about文件夹，里面包含一个index.md文件。

编辑页面内容：

—title: 关于我—

我是一名热爱技术的开发者，喜欢探索新技术和解决问题。

在导航栏中添加链接： 修改主题的配置文件（如themes/landscape/_config.yml），在menu部分添加自定义页面的链接：

menu:

Home: /

About: /about/
